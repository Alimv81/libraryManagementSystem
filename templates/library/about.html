Library management system about page

this webapp aims to serve the users a free management system and private environment for managing daily tasks. having features such as checking a task as done, viewing done or not done tasks, deleting tasks, updating tasks and can store each task with its title, description, deadline date and a flag for being done status. built and developed by django backend framework.

note that if you wish to borrow a book with an account that is signed up as a writer, it won't be possible, and you will have to sign up an second account as member
